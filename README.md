# 🧮 Factorial Calculator

A simple and user-friendly **Factorial Calculator** built using **Python** and **Tkinter**. This desktop application allows users to calculate the factorial of a positive integer through an interactive graphical user interface (GUI).

## ✨ Features

* Easy-to-use GUI
* Calculates factorial of positive integers
* Input validation
* Prevents negative numbers
* Displays warning messages for invalid input
* Supports the **Enter** key to calculate
* Includes a Windows executable (`Factorial Calculator.exe`)

## 🛠️ Technologies Used

* Python 3
* Tkinter

## 📁 Project Files

* `factor.py` – Python source code
* `Factorial Calculator.exe` – Windows executable
* `fa.ico` – Application icon
* `README.md` – Project documentation

## 🚀 How to Run

### Option 1: Run the Executable (Windows)

1. Download the project.
2. Double-click **Factorial Calculator.exe**.

### Option 2: Run the Python Source Code

1. Install Python 3.
2. Open a terminal in the project folder.
3. Run:

```bash
python Factorial_Calculator.py
```
